animateText(text, optionsopt)

Animate this active tile's text.

Parameters
Name Type Attributes Description
text string | Array.<string> | animateTextFunction

If this is a string, the active tile's text will animate through each character one-by-one. If this is an array of strings, the active tile's text will animate through each string one-by-one.
